\section*{Постановка}
Рассматривается задача
\[
f(x)\to \inf,\qquad x\in X=\{x\in E^1:\ x\in X_0,\ g(x)\le 0\}.
\]
Для численного решения использованы два варианта функции нагружения:
\[
\Phi_{13}(t,x)=L\max\{f(x)-t,0\}+MP(x),
\qquad
\Phi_{32}(t,x)=L|f(x)-t|+MP(x),
\]
где
\[
P(x)=\max\{g(x),0\},\qquad L>0,\quad M>0.
\]
Функция
\[
\rho(t)=\inf_{x\in X_0}\Phi(t,x)
\]
строилась численно, после чего искался минимальный корень уравнения
\[
\rho(t)=0.
\]

\section*{Принятые допущения}
В формулировке упражнения для пунктов а), б), г), д) перечислено по несколько вариантов ограничений и множеств \(X_0\). Для численного эксперимента были выбраны следующие постановки:
\begin{itemize}
    \item а) \(f(x)=\arctan x,\ g(x)=x^2-4,\ X_0=\{x\ge 1\}\);
    \item б) \(f(x)=x\sin x,\ g(x)=x^2-1,\ X_0=\{x\ge 0\}\);
    \item г) \(f(x)=1\) при \(x\le 1\), \(f(x)=x^{-1}\) при \(x>1\), \(g(x)=x-1,\ X_0=E^1\);
    \item д) \(f(x)=1,\ g(x)=x,\ X_0=E^1\).
\end{itemize}

Для случаев \(X_0=E^1\) вычисления проводились на усечённом интервале \([-4,4]\), чего достаточно для выбранных примеров и построения графиков.

\section*{Численная схема}
Использовались параметры
\[
L=1,\qquad M=20.
\]
Внутренняя минимизация по \(x\) реализована сеточным поиском на равномерной сетке. Затем на сетке по \(t\) вычислялась функция \(\rho(t)\). Минимальный корень \(t_*\) определялся как наименьшее значение \(t\), для которого выполнено \(\rho(t)\le \varepsilon\), где \(\varepsilon=10^{-4}\).

Из таблицы видно, что оба варианта функции нагружения дают близкие значения минимального корня. Для задач а) и б) получены значения, совпадающие с точными минимумами с точностью до шага сетки. Для задач г) и д) минимум по \(x\) не единственный, поэтому численный алгоритм возвращает один из допустимых представителей.

\section*{Вывод}
Для выбранных постановок численная реализация метода нагруженных функций отработала устойчиво. Оба варианта \(\Phi\) приводят к практически совпадающим значениям \(t_*\), а различия определяются дискретизацией по \(x\) и \(t\). В задачах с неединственным минимумом метод корректно находит допустимую точку, удовлетворяющую условию \(\rho(t_*)\approx 0\).
